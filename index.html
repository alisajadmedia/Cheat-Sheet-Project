# Name Description
1 tr The table row element, is used to add rows to a table before adding table data and table headings.
2 td The table data element, can be nested inside a table row element, to add a cell of data to a table.
3 thead The table head element, defines the headings of table columns encapsulated in table rows.
4 tbody The table body element, is a semantic element that will contain all table data other than table heading and table footer content.
5 th The table heading element, is used to add titles to rows and columns of a table and must be enclosed in a table row element
6 tfoot The table footer element, uses table rows to give footer content or to summarize content at the end of a table.
7 colspan used to span columns in a table
8 rowspan used to span rows in a table
